# AdminPortal

## Dependencies
* Python 3.7
* Django
* Django-Allauth
* Crispy-Forms

Installing dependencies in terminal:
```
pip install django-allauth
pip install django-crispy-forms
```

## Download

```
git clone https://github.com/Brianm0823/AdminPortal.git
```

## Run
In terminal:
```
python manage.py runserver
```
